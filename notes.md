# Занятие 6. PostgreSQL для администраторов баз данных и разработчиков
Свойства транзакций в SQL:
* Атомарность -- применяется всё или не применяется ничего
* Согласованность -- БД переводится из одного согласованного состояния в другое
* Изолированность -- одновременное выполнение транзакций не приводит к взаимному влиянию
* Долговечность -- после фиксации транзакции результат всегда сохраняется

WAL -- write ahead logging
